# Categorical Meta-Prompting Skills

18 Claude skills implementing the categorical meta-prompting framework from [manutej/categorical-meta-prompting](https://github.com/manutej/categorical-meta-prompting).

## Core Implementation Skills

| Skill | Description |
|-------|-------------|
| **effect-ts-ai** | @effect/ai integration patterns for categorical AI composition, typed error handling |
| **dspy-categorical** | DSPy compositional prompt optimization with categorical signatures |
| **lmql-constraints** | LMQL constraint-guided generation DSL for type-safe prompting |
| **langgraph-orchestration** | LangGraph stateful multi-agent graphs with categorical coordination |
| **voltagent-multiagent** | VoltAgent multi-agent system design with natural transformation coordination |

## Installation

Each `.skill` file is a zip archive. To install:

1. Download the `.skill` file
2. Upload to Claude's skill system
3. The skill will be available for triggering based on its description

## Categorical Foundations

These skills implement concepts from:

- **Gavranović et al. (ICML 2024)**: Categorical Deep Learning
- **de Wynter et al. (2025)**: On Meta-Prompting  
- **Bradley (2021)**: Enriched Category Theory of Language
- **Spivak-Niu**: Polynomial Functors for Interaction

## Framework Alignment

These skills support the CC2.0 research workflow:

1. **OBSERVE** → Use arxiv-categorical-ai for paper analysis
2. **REASON** → Apply quality-enriched-prompting for assessment
3. **CREATE** → Use prompt-dsl for compositional prompts
4. **ORCHESTRATE** → Deploy langgraph-orchestration for multi-agent
5. **LEARN** → Extract patterns with polynomial-functors
6. **VERIFY** → Validate with categorical-property-testing
7. **DEPLOY** → Integrate via mcp-categorical
